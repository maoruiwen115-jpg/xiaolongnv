# ai-news

## ⚠️ 最重要的规则

**搜索是第一步，绝对不能跳过。**

日报的AI新闻部分，绝对不能写"今日网络搜索暂时受限""暂沿用近期主题""新闻暂缺"——这是严重质量事故。

如果 Tavily 搜索失败，立即换搜索维度，不能放弃。

---

## 精神

**不是收集信息，是判断哪些信息值得进入日报。**

每天AI新闻海量产出，95%是噪音。这个skill的职责是：找到真正重要的3条，配上来源和分析。

---

## 搜索命令（已验证正确）

**Tavily 搜索**（唯一可用方式）：
```bash
python3 /tmp/tavily_search.py "搜索关键词"
```

**注意**：`openclaw tools search` 命令不存在，不要用。

**搜索示例**：
```bash
# 每日必搜：今日AI新闻
python3 /tmp/tavily_search.py "AI 人工智能 最新新闻 2026年3月"

# 补充维度（根据当日热点选）
python3 /tmp/tavily_search.py "OpenAI 大模型 最新 2026"
python3 /tmp/tavily_search.py "人形机器人 具身智能 2026年"
python3 /tmp/tavily_search.py "AI 商业化 应用 落地 2026"
python3 /tmp/tavily_search.py "算力 芯片 GPU 涨价 2026"
```

## 搜索流程

### Step 1：主搜索
```bash
python3 /tmp/tavily_search.py "AI 人工智能 最新新闻 $(date +%Y年%m月%d日)"
```

### Step 2：补充搜索（如需要）
根据 Step 1 结果，选择相关维度继续搜，例如：
- 如果有 OpenAI 新闻 → 搜更多 OpenAI 细节
- 如果有机器人新闻 → 搜具身智能
- 如果有商业化新闻 → 搜 AI 应用

### Step 3：验证
每条新闻必须确认：
- [ ] 有真实 URL
- [ ] 日期是今天或昨天
- [ ] 不是广告/软文

## 输出格式

```markdown
### 1. [标题] 🔥必须看
**来源**：[媒体名](URL) | 日期
**摘要**：一句话描述
**深度解读**：
1. [维度1]：...
2. [维度2]：...
**为什么重要**：对用户/行业的影响
```

## 质量标准

### 来源优先级
1. 官方公告（OpenAI/Anthropic/Tesla官网）
2. 权威媒体（财联社/虎嗅/36氪/澎湃/每日经济新闻）
3. 技术媒体（HuggingFace blog/VentureBeat）

### 选文标准（3条原则）
- **🔥必须看**：有重大突破或格局变化，影响未来3个月
- **⭐推荐**：有价值但时效性/影响范围稍小
- **不选**：产品发布/人事变动/常规更新

### ⚠️ 绝对禁止
- ❌ "今日网络搜索暂时受限"
- ❌ "暂沿用近期核心主题"
- ❌ "AI新闻暂缺"
- ❌ 新闻没有 URL

### 重要信息的交叉验证流程
对于重要结论（如"清明户外降低皮质醇"）：
1. **先搜学术来源确认** → PubMed/Google Scholar/PNAS/Nature/Science
2. **再用大众来源补充背景** → 36氪/虎嗅/机器之心
3. **如果学术来源找不到** → 才用大众来源

**搜索命令**：
```bash
# 新闻搜索（主力）
python3 /tmp/tavily_search.py "AI 人工智能 最新新闻 $(date +%Y年%m月%d日)"

# 重要信息交叉验证（学术优先）
python3 /tmp/tavily_search.py "cortisol green space nature park stress reduction PubMed"
python3 /tmp/tavily_search.py "清明 户外 绿色植被 皮质醇 学术研究"
```
