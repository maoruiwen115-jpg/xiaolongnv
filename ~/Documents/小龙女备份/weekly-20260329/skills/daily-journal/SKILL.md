## 科学依据格式（重要更新 2026-03-27）

**旧格式（已废弃）：**
```
- 结论文字 [^1]

底部脚注：
[^1]: [标题](URL)
```

**新格式（当前使用）：**
```
- [清明时节户外绿色植被可降低皮质醇水平，显著改善情绪](https://URL) ↗
- [规律饮菊花茶可改善肝火旺盛导致的头痛和烦躁](https://URL) ↗
```

**格式规则：**
- 每条科学依据后直接跟内嵌链接，末尾加 `↗`
- **不要脚注区域**（底部不再放参考文献）
- 链接文字是研究结论的简短标题
- 按课题分组，每个课题3条依据

## AI新闻格式

```
### 1. 新闻标题 🔥必须看 / ⭐推荐

**来源**：[媒体名《文章标题》](URL) | YYYY-MM-DD

**摘要**：2-3句话核心内容

**深度解读**：
1. **维度一**：...
2. **维度二**：...
3. **维度三**：...

**为什么重要**：1句话总结
```

**AI新闻必须包含：**
- [ ] 每条带完整可点击URL
- [ ] 标注"🔥必须看"或"⭐推荐"
- [ ] 有深度解读（为什么重要）
- [ ] **绝对禁止写"网络搜索受限/暂沿用/新闻暂缺"**

**搜索失败时：** 换关键词继续搜，不放弃 → 直到搜到为止
- 如果确实无法获取新闻，标注"本期AI新闻待完善"而不是"网络受限"
- **不允许跳过搜索步骤**

## 6课题研究优先级（权威优先，2026-03-28 更新）

每个课题输出前，按此顺序找依据：

| 优先级 | 来源 | 适用场景 |
|--------|------|---------|
| 1 | 学术期刊（PubMed/Google Scholar/PNAS/Nature/Science） | 机制/原理/重大发现 |
| 2 | 权威机构（Harvard Medical School/WHO/卫健委/ACSM） | 健康指南/运动处方 |
| 3 | 专业书籍（中医经典/营养学/心理学教材） | 节气养生/长期积累 |
| 4 | 大众媒体（36氪/虎嗅/机器之心） | 补充背景/新闻关联 |

**交叉验证流程**：
1. 核心结论 → 先找学术来源
2. 学术找不到 → 用权威机构来源
3. 都找不到 → 才用大众媒体
4. 每条依据格式：`[结论](URL) ↗`

## 常见错误警示

| 错误 | 防范 |
|------|------|
| 科学依据用脚注格式 | 用 `[结论](URL) ↗` 内嵌格式 |
| 底部有脚注区域 | 结语后直接是小龙女印记，不要参考文献区 |
| AI新闻没URL | 每条新闻必须带完整可点击URL |
| 写"洋洋"等错误人名 | 先读 MEMORY.md 确认家庭成员 |
| AI新闻写"暂沿用" | 搜索失败换关键词继续，不放弃 |
| 标题只有人名没有关键词 | 参考：`YYYY-MM-DD-日报-关键词1·关键词2·关键词3` |
