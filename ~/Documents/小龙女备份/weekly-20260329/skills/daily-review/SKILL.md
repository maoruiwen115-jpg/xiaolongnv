# daily-review

晚间复盘，引导用户对今天6个课题的执行情况进行反思。

## 复盘格式

```
🌙 今晚复盘 | YYYY-MM-DD

## 6课题今日完成情况

### 🏃 健康
- 早起：[几点]
- 锻炼：[做了哪些]
- 八段锦：[做了吗]
- 睡眠：[几点睡]
- 感受：...

### 📚 成长
- 阅读：[读了什么]
- AI学习：[学了什么]
- 输出：[有没有]
- 感受：...

### ⚡ 能量
- 上午黄金窗口利用：[好/一般/差]
- 下午能量低谷管理：...
- 总体状态：...

### 💼 事业
- 今日最重要的一件事：...
- AI提效用了什么：...
- 感受：...

### 💝 关系
- 洋洋/当当/麦兜陪伴：...
- 晚饭后散步：...
- 感受：...

### 🙏 感恩
- 今日感恩的事：...
- 写日记了吗：...

---

## 明日最重要的一件事
...

## 明日改进点
...

## 今日最感恩的瞬间
...
```

## 执行流程

1. 询问用户今天各课题的执行情况
2. 记录用户反馈（即使没有全部完成也不要紧）
3. 引导用户思考：明日最重要的1件事是什么？
4. 记录反馈到 `memory/YYYY-MM-DD.md`
5. 感谢用户的分享

## 注意事项

- 不要评判用户完成度，专注引导反思
- 用"今天哪件事让你觉得最有收获"这种开放式问题
- 结束时简短肯定，不冗长
