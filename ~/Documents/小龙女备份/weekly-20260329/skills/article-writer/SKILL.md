# article-writer

## 精神

**爆款文章不是信息堆砌，是让人看完觉得"说的就是我"。**

结构是骨架，细节是血肉，情感是灵魂。三者缺一不可。

---

## 文章结构（固定模板）

### 标题（最重要，决定打开率）

**爆款标题公式**：`人物 + 最劲爆的数字/事件 + 结果`

**示例**：
- "苏轼在监狱待了103天，几近处死。他后来发明了东坡肉，写出《赤壁赋》。"
- "56岁创业，被所有人嘲笑：他用一辈子做好一件事，改变全球科技史"
- "31岁蹬三轮，80岁捐全部财产：这个河北老人教我什么是真正的富"

**标题检查**：
- [ ] 有具体数字（年龄/年份/次数）
- [ ] 有冲突/反差（顺境→逆境 或 逆境→成功）
- [ ] 有悬念（让人想点进去）

---

### 开头（3秒抓住注意力）

**爆款开头公式**：先抛**最劲爆的冲突/反转**，再展开

**不要**：
- "今天我想讲一个故事"
- "最近发生了一件事"
- "我想跟大家分享一个人物"

**要用**：第一句话就扔出一个让读者震惊的事实

**示例（苏轼版）**：
```
43岁那年，苏轼差点死了。

不是夸张。

1079年，因为几首诗被举报"讥讽朝政"，他从堂堂杭州太守，被押送进京，关进监狱。
审了103天。
```

---

### 正文（故事驱动，4个模块）

**模块1：跌到谷底**
- 用具体数字讲最惨的时刻
- 有细节：当事人的处境、心理、对话
- 制造"这个人完了"的感受

**模块2：转折时刻**
- 他做了什么不一样的事？
- 不是"努力坚持"，而是具体的行动
- 给出原话/金句

**模块3：反驳常识**
- "你可能会说：XX，我们普通人比不了"
- 真的吗？——反驳这个想法
- 给出普通人能学的具体方法

**模块4：给困境命名**
- 他给困境取名了吗？（苏轼叫"东坡"）
- 你的"黄州"是什么？
- 给出行动指引

**段落规则**：
- 一个段落 = 一个信息
- 有细节：具体说了什么/发生了什么
- 有数字：年龄/年份/金额
- 有对话：当事人的原话
- **叙述段 + 反思段交替**，不要连续3段纯叙述

---

### 结尾（Callback + 行动）

**Callback**：回扣开头，形成闭环

**示例（苏轼版）**：
```
开头问：43岁差点死了，这辈子完了吧？
结尾答：不是。他种地、做饭、写文章、陪孩子，还顺手创造了东坡肉。
```

**行动**：一句话总结核心启示
- **不要**："要乐观面对困难"（太鸡汤）
- **要**："他从来不问'为什么是我'，问的是'现在，我能做什么？'"

## 写作自检清单

| 检查项 | 标准 |
|--------|------|
| 标题有数字+冲突 | 打开率的关键 |
| 开头第一句是劲爆事实 | 不是"今天我想讲..." |
| 模块1有跌到谷底 | 制造"完了"的感觉 |
| 模块3反驳常识 | 有"真的吗？"的反问 |
| 模块4给困境命名 | 苏轼叫"东坡"，你呢？ |
| 有具体数字/细节/对话 | 不是泛泛而谈 |
| 结尾回扣开头 | 形成闭环 |
| 行动不是鸡汤 | 要具体，不要"要乐观" |
| 有书籍推荐 | 2-3本，有星级，有人群 |
| 有来源注释 | "本文人物经历参考XX" |
