# writing skill - 小龙女的写作教练

## 精神

写作是思考的外化。写不好，是因为想不清楚。

小龙女的目标：成为一个能写出有价值的、有人读的、服务他人的内容的AI。

---

## 核心原则（来自11篇文档的精华）

### 1. 创作的目的是沟通，不是表达
- **定义**：沟通是让对方明白他之前不明白的道理
- **起点**：想清楚你要让读者获得什么感知
- **终点**：读者读完后的认知变化

### 2. 输入质量决定输出质量
- 输入是垃圾，输出必然是垃圾
- 检索式阅读：search → search → research
- 选书是自己的能力，不断提升选择标准

### 3. 写什么比怎么写重要1万倍
- 写读者不知道的认知增量
- 提供"以前没有提供过的东西"
- 写只有你能写好的内容

### 4. 四大修辞手法（优先级：思想 > 文采）
| 手法 | 核心 | 案例 |
|------|------|------|
| 具象 | 具体具体再具体 | 名词动词要精准 |
| 排比 | 信息密度高 | 三点并列 |
| 类比 | 从已知到未知的桥梁 | X≈A→讲A→X不言自明 |
| 隐喻 | 深层连接 | 围城/枪炮 |

### 5. 三种逻辑关系
- **并列**：和、或、以及（思考丰富）
- **递进**：更、甚至、竟然（思考深入）
- **转折**：但是、然而、却（思考全面）

**穿透力最强的两种模式**：
- 递进-递进（层层深入）
- 递进-转折（高潮反转）
- 转折-递进（先破后立）

### 6. Don't tell, show
- 传递感知，不要只告诉结论
- 展示过程，展示细节
- 让读者自己得出结论

### 7. 逻辑三基本功
- **分类**：排他性 + 完整性
- **比较**：同一范畴 + 相同属性
- **因果**：A是B的理由？唯一？最重要？

### 8. 长期主义
- 抄袭/搬运：短期有效，长期等于零
- 原创：短期难，长期最优解
- 急功近利必翻车

## 微信公众号爆款逻辑

### 标题4U原则
| 原则 | 说明 | 案例 |
|------|------|------|
| **Useful** | 对读者有用 | "3个方法让你..." |
| **Urgent** | 紧迫感 | "今天必须..." |
| **Ultra-specific** | 极度具体 | "103天+结果"比"很久"好 |
| **Unique** | 独特角度 | "治愈了后世一千年" |

### 开头3秒法则
- 前3秒必须抓住读者
- 不能铺垫太久
- 直接抛问题/痛点/冲突

### 节奏：300字一个小钩子
- 每300字内必须有新刺激（新观点/金句/案例/追问）
- 防止读者疲劳

### 金句密度：每400字一个
- 加粗，让人想截图
- 金句是读者分享的"证据"

### 排版：手机阅读友好
| 规范 | 说明 |
|------|------|
| 每段行数 | ≤3行（大量留白） |
| 段落之间 | 空一行 |
| 金句 | 加粗 |
| 关键数字 | 加粗 |
| 问句 | 单独成段，引起注意 |

### 结尾：情感共鸣 + 召唤行动
- 温暖接地气
- 留讨论话题（引导评论区互动）
- 不要只给延伸阅读，要问读者一个问题

### 叙事 vs 说理比例
- 50%叙事，50%说理（人物传记类）
- 叙事要为说理服务，不是为了讲完故事

## 排版规范（章节结构）

**公众号文章标准结构**：

```
## 前言
- 用痛点/冲突/问题开头
- 直接把读者拉进场景
- 3秒内抓住注意力

## 01
- 第一个论点/故事

## 02
- 第二个论点/深化

## 03
- 第三个论点/再深化

## 04（如需要）
- 第四个论点/翻转

## 05（如需要）
- 第五个论点/最后深化

## 结语
- 核心道理（一句话）
- 召唤行动/讨论话题
- 延伸阅读（可选）
```

**章节命名规则**：
- 用"前言"开头，不用"0"
- 章节用 01/02/03...，不用 01混02 的写法
- 最后用"结语"收尾

## 写作自检清单（每次写完必查）

### 内容自检
- [ ] 我要让读者获得什么认知/感知？
- [ ] 读者读完后会有什么变化？
- [ ] 核心论点是什么？（一句话能说清楚吗？）
- [ ] 有没有类比/隐喻让复杂变简单？
- [ ] 金句在哪里？（至少每400字一个）
- [ ] 叙事和说理的比例合适吗？
- [ ] 有没有在"告诉"而不是"展示"？

### 排版自检
- [ ] 章节结构清晰吗？（前言 → 01 → 02 → ... → 结语）
- [ ] 每段≤3行？
- [ ] 段落之间有空行吗？
- [ ] 金句/关键数字加粗了吗？
- [ ] 结尾有讨论话题吗？

### 事实自检
- [ ] 所有史实有据可查吗？
- [ ] 细节是真实还是推测？（推测需要标注）
- [ ] 引用有来源吗？

## 人物传记爆款结构

### 五步结构
```
## 前言
- 用人物最震撼的一个事实/数字开头
- "如果你是我，你会怎么办？"

## 01：他做了什么？（第一层）
- 叙事 + 金句

## 02：我们对比（第二层）
- 普通人的选择 vs 他是怎么想的
- 用排比展示"我们总在等"

## 03：他怎么做到的（第三层）
- 不是乐观，是想明白了一件事
- 核心哲学

## 04：更深层（第四层）
- 递进/翻转
- 展示更多细节

## 05：联系到读者（第五层）
- 清明反思 / 带孩子的场景
- 联系到读者自己的生活

## 结语
- 一句话核心认知
- 讨论话题
```

### 爆款标题公式
```
[人物] + [最大挑战/反差] + [结果/升华]

案例：
"苏轼被贬3次，从太守贬到天涯海角。65岁病逝前，他写了一首诗，治愈了后世一千年。"

公式拆解：
- 人物：苏轼
- 最大挑战/反差：被贬3次，从太守到天涯海角
- 结果/升华：65岁病逝前，治愈了后世一千年
```

## 写作skill执行清单

每次写任何内容前，逐项确认：

### 构思阶段
- [ ] 我要让读者获得什么认知/感知？
- [ ] 这个内容只有我能写吗？
- [ ] 读者读完后会有什么变化？
- [ ] 核心论点一句话能说清楚吗？

### 结构阶段
- [ ] 章节结构用 前言 → 01 → 02 → ... → 结语 吗？
- [ ] 用什么逻辑关系？（并列/递进/转折）
- [ ] 有没有具象的案例/细节？
- [ ] 有没有类比/隐喻让复杂变简单？

### 表达阶段
- [ ] 是否在"告诉"而不是"展示"？
- [ ] 名词/动词是否精准？
- [ ] 有没有多余的副词/形容词？
- [ ] 金句在哪里？

### 审核阶段
- [ ] 逻辑通顺吗？
- [ ] 有没有更好的类比？
- [ ] 读者能理解吗？
- [ ] 排版符合规范吗？
- [ ] 事实有据可查吗？

## 自我训练计划

### 每日练习
- [ ] 圈出今天读的文章的名词/动词/形容词
- [ ] 分析一篇文章的逻辑结构（并列/递进/转折）
- [ ] 找一个类比，把今天学的东西讲给别人听
- [ ] 检查自己写的内容有没有"告诉"而不是"展示"

### 每周练习
- [ ] 写一篇300字的学习笔记
- [ ] 找一个类比，写进笔记
- [ ] 用自检清单检查
- [ ] 请用户反馈哪里可以更好

### 每月复盘
- [ ] 这个月写了多少字
- [ ] 哪些写得好？哪些需要改进？
- [ ] 下个月要专注哪个维度？
- [ ] 用户的反馈有什么共性？
