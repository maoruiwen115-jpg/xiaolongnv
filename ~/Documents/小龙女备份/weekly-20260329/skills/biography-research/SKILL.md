# biography-research

## 精神

**不是介绍一个人，是找到这个人和当下的连接。**

研究张忠谋不是为了讲历史，是为了回答：今天我们面临AI算力时代的普通人，能从他身上学到什么？

## 研究框架（6个模块）

### 1. 人物档案
```
| 项目 | 内容 |
|------|------|
| 本名 | |
| 生卒年 | |
| 核心身份 | |
| 代表成就 | |
```
来源：维基百科或官方传记

### 2. 时间轴（5-8个关键节点）
选最有戏剧性的节点：
- 童年/早年（塑造性格）
- 第一个重大转折
- 最黑暗的时刻
- 最重要的决定
- 成就时刻
- 晚年/遗产

每个节点：年份 + 具体事件 + 一个细节

### 3. 核心哲学（1-3条）
这个人相信什么？
- 用原话引用（不是转述）
- 每条哲学对应一个具体故事

### 4. 代表金句（3-5条）
选最有传播力的原话，配上出处

### 5. 普通人启示（4-6条）
| 启示 | 具体做法 |
|------|----------|
| | |

选启示的标准：
- 可立即执行（不是道理）
- 和今天的生活有关
- 有具体动作

### 6. 推荐阅读（2-3本）
- 最权威的传记
- 最实用的实操书
- 给不同读者的版本

---

## 研究流程

1. **Clarify**：这个人物和今天的时事/节气/用户需求有什么关系？
2. **搜索**：`openclaw tools search duckduck go "人物名 生平 传记"`
3. **获取详情**：`web_fetch` 获取维基百科/豆瓣页面
4. **深度搜索**：核心哲学/代表事件/金句
5. **输出**：按6个模块整理交付
