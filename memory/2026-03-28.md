## 李笑来写作课第一课（文字本质）
- 词汇 → 句子 → 段落 → 文章
- 名词（感知事物）、动词（感知动作）、形容词（对事物的感知）、副词（对感知的感知）
- 核心：所有创作都是关于感知；你只能描述你能感知的事物
- 作业：每天圈名词、画形容词、分析文章至少7年

## 李笑来写作课第七课：逻辑模板（补充）
- 元素间三种逻辑关系：并列（和/或/以及）、递进（更/甚至/竟然）、转折（但是/然而/却）
- 三种最强排列模式（穿透力最强）：递进-递进、递进-转折、转折-递进
- 好故事模板：小人物+大目标；第一幕困难→解决→第二幕更大困难→解决→第三幕绝境→反转→成功
- 作业：分析《把时间当做朋友》和《定投改变命运》中的并列、递进、转折关系

## 李笑来写作课第八课：创作的目的和目标（补充）
- 核心：创作的目的不是"表达"，而是"沟通"
- 沟通的定义：让对方明白他之前不明白的道理
- 创作四阶段：思考能力 → 表达能力 → 价值观 → 影响力
- 有效沟通需要提前做大量功课（80%是别人看不见的工作）

## 李笑来写作课·个人品牌建设（补充）
- 个人品牌 = 名，个人品牌变现 = 利
- 本质古往今来都一样，新瓶装旧酒
- **核心1**：个人品牌是别人对你的看法总和（不是你以为的你）
- **核心2**：靠内在不靠外在（颜值终将凋零，内在可以无限积累）
- **核心3**：长期思维（急功近利短期有效，长期必翻车）
- 方法论：做服务者不做领导者；提供真正经过验证的对别人有用的东西；建社群
- 金句："绝大多数人只在意流量，而流量本身不是钱，服务才是钱"

## 果麦编辑王光裕分享（出版情报）
- 好作品不会被埋没（科技进步让信息流转极快）
- 出版社的敌人是王者荣耀/抖音
- 投稿前必须想清楚：一句话/三分钟能讲清楚；"只有我能写好"
- 与编辑的合作：平等关系；不要教编辑做书/营销；自媒体先发表=提前接受市场检验
- 出版流程：投稿 → 选题会 → 签订合同 → 编辑 → 书名+文案 → 装帧设计 → 出版社审稿 → 印刷 → 发行
- 如何判断编辑靠不靠谱：入行是否满三年；问设计师/发行同事；信任你的编辑
- 好书标准：先分对错，再说美丑（简单、清晰、主题明确）
